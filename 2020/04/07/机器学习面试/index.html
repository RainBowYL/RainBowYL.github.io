机器学习面试

发表于 2020-04-07 | 分类于 机器学习

机器学习

bias 与 variance:

bias : 平均预测结果与实际预测结果的差异； bias高，不够关注训练数据，模型简单，易造成欠拟合

variance：模型输出在一个数据点上的分布（或聚类）。方差大，可能密切关注训练数据，但泛化能力弱，过拟合

两者无法同时满足，只有追求两者之间的权衡。

分类

问题的指标：准确率未必可靠，可以用混淆矩阵，ROC曲线(伪正率横轴，真正率负轴)，AUC值

激活函数的作用：加入非线性因素提高神经网络模型的表达能力，解决线性问题所不能解决的问题

数据不平衡问题：

（1）选择合适的评价指标：精确度、召回率、F1分数、AUC等指标

准确率：所有样本中预测正确的比例

精确率：预测为正类的样本中预测正确的比例

召回率：实际为正类的样本中预测正确的比例

F1分数（精确率和召回率的调和平均数）：\(F_{1}= 2 * \cfrac{recall*precision}{recall+precision}\)

（2）对数据集进行重采样：欠采样和过采样

（3）集成多个不同的模型：比如有1000个罕见样本和10000个常见样本，可以选择10个模型，每次训练1000个罕见样本和1000个常见样本，而后使用集成学习技术得到最佳结果

（4）重新设计模型---成本函数：针对数据丰富的类别加入惩罚技术，是损失函数可以覆盖所有类别

SVM

（1）核方法：将低维空间的数据映射到高维空间，使其线性可分 :\(f(x)=\phi(x)^T\phi(x)\)核函数的内积

（2）松弛变量\(\epsilon\)和惩罚因子\(C\):将线性可分SVM拓展为线性不可分SVM，分类错误才会有松弛变量，惩罚因子对线性不可分点的惩罚

增大C，泛化能力变弱，无穷大时，退化成硬间隔线性可分；减小C，模型泛化能力变好，为0时，模型无意义

“离群点”才会有松弛变量，值越大，离群点越远

（3）缺失数据敏感：没有处理缺失值的策略，希望样本特征空间线性可分，特征空间的好坏对SVM影响很大

（4）如何选择核函数：特征维数d样本数m

d>m,线性核（文本分类问题）；d比较小，m中等，使用RBF核。d小，m特别大，不如神经网络

（5）SVR：损失函数：假设能容忍\(f(x)\)和\(y\)之间有一定偏差，当\(f(x)\)和\(y\)之间的偏差大于该值时才计算损失

（6）原始问题转化成对偶问题：a.更易求解，原问题复杂度与特征维度有关，转换成对对偶问题与支持向量个数有关 b.更方便地引入核函数 c.对偶问题将约束问题转换成了等式约束 d.更加高效

LR

（1）LR可以用核函数吗：可以，低维线性不可分的时候，应用到高维空间，解决非线性问题

（2）为什么LR需要归一化或者取对数，为什么LR特征离散化后效果更好，为什么把特征组合之后还能提升？

​ 增强了特征的表达能力，或者更容易线性可分

（3） 梯度下降

L1和L2正则化

（1）L1：容易得到稀疏解（0比较多）

​ L2 : 解比较平滑，保证解中接近于0的维度多，降低模型复杂度

（2）L0， L1， L2正则化的推导，L1， L2正则化的使用场景，以及为什么L1， L2会有稀疏解

聚类

K-means 优点：容易实现 缺点：可能收敛于局部最小值，在大规模数据上收敛慢 优化方法：二分K-均值（MSE最小的族）

KNN近邻算法： kd 树， 方差最小，便于数据集中，选取中心，便于二分查找

DBSCAN基于密度的聚类算法：可达距离（每组联通的核心点组成一个cluster，将每个边界指派到与之相关的cluster中）

Adaboost 与GBDT的区别

adaboost：调整权重，尽量向梯度方向减小

GBDT：计算负梯度，每一轮向损失函数减小的方向变化(关注残差去拟合残差，可使用更多种类的目标函数)

boosting 与 bagging 算法比较（RF与GBDT的区别，前者降低模型方差，后者偏差

Bagging boosting
样本选择 随机放回抽样 训练集不变
权重 均匀取样，样本权重相等 调整错分样本权重
预测函数 权重相等 误差越小，权重越大
并行计算 预测函数可并行计算 按顺序迭代计算
最终结果 多数表决（回归平均取值） 加权融合
异常值敏感性 不敏感 比较敏感
泛化能力 不易过拟合 易过拟合

LR 与GDBT的区别

LR：线性模型，可解释性很强，容易并行化，但学习能力有限，需要大量的人工特征

GBDT：非线性模型，特征组合优势，特征表达能力强，无法并行训练，容易过拟合

Xgboost

与GBDT的区别：

损失函数 二阶展开（精准、支持自定义）
正则项 防止过拟合（L2正则化）
子，列采样 与随机森林类似，防过拟合
缺失值处理 划入默认分支（自动学习每个非叶子节点的默认方向）
并行化 特征维度并行，按值排序，存储为块结构，对每个block多线程并行查找寻找最佳分割

决策树剪枝算法：从叶节点向上回溯，尝试对某个节点进行剪枝，比较预测函数，最后通过动态规划得到全局最优的剪枝方案

决策树

决策树处理连续值的方法：离散化（LightGBM中将连续值映射到bins中，将bins看成是离散变量。一般的决策树是离散二值化）

CART（回归用平方误差最小化原则，分类树用基尼指数最小化准则）

信息熵和基尼指数的关系：信息熵在x = 1处的一节泰勒展开就是基尼指数

梯度下降算法

随机梯度下降算法(SGD)：每次只随机选取一个样本进行梯度下降，更新权重

优点：训练速度快；缺点：准确度下降，并不是全局最优；不易于并行实现。

批量梯度下降算法：每次选取一个批次的数据去更新权重

优化器：

ADam：损失函数对每个参数的梯度的一阶矩估计和二阶矩估计动态调整

Adagard：自适应学习率，设置一个全局学习率，会自动调整
